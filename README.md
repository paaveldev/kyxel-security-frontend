# **Kyxel Security — Frontend**

This repository contains the official frontend for **Kyxel Security**, part of the Kyxel ecosystem. It includes the public interface, visual components, and informational sections related to our cybersecurity and infrastructure-protection services.

## **Technologies Used**

* **Astro** – Fast, lightweight framework for static and component-based sites.
* **Tailwind CSS** – Utility-first styling for a clean and responsive layout.
* **TypeScript** – Strong typing for safer development.
* **Node.js / npm** – Dependency management and project tooling.

## **Key Features**

* Fully responsive design.
* Modular and reusable UI components.
* Smooth animations and transitions.
* Fast loading and clean structure.
* Organized file architecture for easy maintenance.

## **Installation**

1. Clone the repository:

   ```sh
   git clone https://github.com/paaveldev/kyxel-security-frontend.git
   ```
2. Install dependencies:

   ```sh
   npm install
   ```
3. Start the development server:

   ```sh
   npm run dev
   ```
4. Open in your browser:

   ```
   http://localhost:4321
   ```

## **Production Build**

```sh
npm run build
```

The optimized output will be generated inside the `dist/` directory.
